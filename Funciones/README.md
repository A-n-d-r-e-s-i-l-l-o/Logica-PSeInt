# Funciones

En esta sección exploro el concepto de **Modularización**. Las funciones permiten dividir un programa complejo en partes más pequeñas, reutilizables y fáciles de mantener.

### Conceptos Aplicados:
* **Funciones con Retorno:** Aquellas que procesan datos y devuelven un resultado al programa principal mediante una variable de retorno.
* **Paso de Parámetros:** Envío de información desde el algoritmo principal hacia la función.
* **Reutilización de Código:** Evitar la repetición de lógica mediante bloques especializados.
