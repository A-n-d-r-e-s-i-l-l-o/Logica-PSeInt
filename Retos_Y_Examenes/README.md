# Retos y Exámenes
En esta carpeta subo los proyectos evaluados de la universidad y retos personales que requieren integrar múltiples conocimientos.

### Destacados:
* **Sistema de Ventas (Parcial 1):** Un programa complejo que gestiona categorías de productos, tipos de clientes (VIP, Mayorista), descuentos automáticos e impuestos (IGV).
* **Calculadora con Menú:** Ejercicio de práctica para dominar el ciclo `Repetir` y la estructura `Segun`.
* **Cajero Automático:** Simulación completa con depósito, retiro y gestión de saldo.
* **Gestión de Promedios:** Sistema que procesa múltiples alumnos, valida notas y determina estados de aprobación.
